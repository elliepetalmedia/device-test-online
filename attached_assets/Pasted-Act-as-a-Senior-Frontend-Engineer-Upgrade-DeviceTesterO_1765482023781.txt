### 2. JAVASCRIPT ROUTER (script.js)

Create a mapping object connecting URL paths to your existing Tab IDs and specific SEO metadata.
Example structure:
const routes = {

**B. Handle Initial Load:**
- On `window.load`, check `window.location.pathname`.
- If the path matches a route, automatically `click()` that tab to open it.
- If no match, default to Home.

**C. Handle Tab Clicks (Update URL):**
- Update your existing "Tab Click" event listener.
- When a user clicks a tab, use `history.pushState(null, '', '/new-path')` to change the URL in the browser bar without reloading.
- Update `document.title` and the `<meta name="description">` content immediately to match the new tool.

**D. Handle Back Button:**
- Listen for the `popstate` event. When the user hits Back, switch to the correct tab based on the previous URL.
